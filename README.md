# Sales Performance Dashboard

End-to-end data analysis project using Python and SQL to uncover business insights from 10,000+ retail orders.

## Problem Statement
A retail company wants to understand which categories, regions, and products are driving profit — and which are losing money.

## Tools Used
- Python (pandas, matplotlib, seaborn)
- SQL (SQLite)
- Tableau / Power BI (dashboard)

## Key Findings
1. **Furniture has only 2.49% profit margin** compared to 17% for Technology and Office Supplies — mainly due to heavy discounting
2. **Central region underperforms** with 7.92% margin, nearly half of the West region (14.94%)
3. **Sales spike every September, November, December** — a clear seasonal pattern across all 4 years
4. **Canon imageCLASS 2200 Copier** is the top product with $25,199 profit
5. **3D Printers are loss-makers** — Cubify CubeX alone lost $8,879, suggesting a serious discount or pricing problem
